EDCG - Ruegen Airport
----------------------------

How to install the scenery?

Copy or unzip all files and put it in the following folder:
Flightgear -> data -> Scenery -> Objects -> e010n50 -> e013n54

Be sure that you replace the stg-file named "3171353" and "3171344" otherwise it won't work later.
